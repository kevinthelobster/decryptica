# SEO & Content Strategy Guide for Decryptica

## Overview
This guide documents best practices for creating content that ranks well on Google and is optimized for AI/LLM discovery (GEO - Generative Engine Optimization).

---

## Part 1: Traditional SEO Best Practices

### 1.1 Keyword Research
- **Primary Keywords**: High-volume terms relevant to your niche
- **Long-tail Keywords**: Specific, lower-volume but higher intent (e.g., "best Solana wallet for NFTs 2026")
- **Semantic Keywords**: Related terms that reinforce topic relevance

### 1.2 On-Page SEO Elements
| Element | Best Practice |
|---------|---------------|
| **Title Tag** | Include primary keyword near start, 50-60 chars |
| **Meta Description** | Compelling summary, 150-160 chars, include CTA |
| **H1** | One per page, include primary keyword |
| **H2-H6** | Use for hierarchy, include secondary keywords |
| **URL** | Short, descriptive, include keyword |
| **Images** | Alt text with keywords, compressed |
| **Internal Links** | Link to related articles |

### 1.3 Content Structure
- **Introduction**: Hook + what reader will learn
- **Body**: Clear headings, scannable format
- **Conclusion**: Summary + CTA
- **Length**: 1,500-3,000 words for competitive topics

### 1.4 E-E-A-T Principles
- **Experience**: Show firsthand experience
- **Expertise**: Demonstrate knowledge
- **Authoritativeness**: Cite sources, build credibility
- **Trustworthiness**: Accurate info, transparent

## Part 2: Generative Engine Optimization (GEO)

### 2.1 What is GEO?
Optimizing content for AI/LLM discovery. As more people use ChatGPT, Claude, Perplexity for answers, getting your content cited by AI is crucial.

### 2.2 Key Strategies

#### A. Structured Data
```html
<schema.org type="Article">
  - headline
  - author
  - datePublished
  - description
  - image
```

#### B. Clear, Factual Content
- Answer questions directly
- Use bullet points and numbered lists
- Include statistics and data
- Cite sources with links

#### C. Authority Building
- Expert quotes and citations
- Original research/data
- Comprehensive "ultimate guides"
- Update content regularly

#### D. Q&A Format
- Include FAQ sections
- Use question-based headings
- Answer "People Also Ask" questions

### 2.3 AI Citation Tips
1. **Be authoritative**: AI prefers credible sources
2. **Use facts**: Include specific numbers, dates
3. **Be comprehensive**: Cover topic fully
4. **Use headers**: Clear structure helps AI parse
5. **Include sources**: Link to authoritative references

## Part 3: Content Writing Template

### Article Structure
```markdown
# [Primary Keyword] - [Secondary Keyword] | Brand

## Meta Description (150 chars)
[Compelling summary with keyword]

## Introduction
- Hook
- What reader will learn
- Why it matters

## H2: [Topic Section 1]
### H3: [Sub-topic]
### H3: [Sub-topic]

## H2: [Topic Section 2]
...content...

## H2: Conclusion
- Summary
- CTA

## FAQ
### Question 1?
Answer...

## References/Sources
- Source 1
- Source 2
```

### SEO Checklist for Each Article
- [ ] Primary keyword in title (start)
- [ ] Primary keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] Meta description with keyword
- [ ] URL contains keyword
- [ ] Alt text on images
- [ ] Internal links to 2+ related articles
- [ ] External links to authoritative sources
- [ ] FAQ section with related questions
- [ ] 1,500+ words for main articles

## Part 5: Publishing Workflow

### 5.1 Before Publishing
1. Run keyword research
2. Check competing articles
3. Create outline
4. Write first draft
5. Add SEO elements
6. Add internal links
7. Proofread
8. Add to sitemap

### 5.2 After Publishing
1. Submit to Google Search Console
2. Share on social (optional)
3. Build internal links from related articles
4. Monitor rankings
5. Update quarterly
